# Comandos de interés
No te preocupes, este espacio es solo para colocar comandos de mi interés para tenerlos a la mano sobre mi proyecto de Python.

´uvicorn app.main:app --reload´

Este comando corre mi aplicación de Python, su estructura es la siguiente:

´carpetas.nombre_archivo_py:app --reload´

El '--reload' significa que la aplicación se recargará con los cambios sin necesidad de correr el comando de nuevo, solo con guardar los cambios.
